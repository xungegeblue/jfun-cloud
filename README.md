jfun-cloud(项目说明)
    
>    一个Spring Cloud项目的脚手架，集成了基本的分布式模块

* 公共模块
    
        公共的方法和资源
    
* 系统模块(jfun-system)

        基本的用户管理、角色管理、权限管理

* 认证中心(jfun-auth)

        OAuth2进行认证
    
* 分布式网关(jfun-gateway)

        使用Zuul
    
* 日志中心(jfun-log)
    
        统一日志消息
    
* 订单业务(jfun-order)
    
        基于最终一致性的订单服务
    
* 秒杀业务(jfun-secKill)
        
        秒杀
